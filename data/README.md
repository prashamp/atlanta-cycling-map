# data/

Generated by `scripts/fetch-data.mjs` — see the repo README's **Loading real
data** section.

When present, these files override the sample data embedded in `index.html`:

| File | Contents | Source |
|---|---|---|
| `infrastructure.js` | `window.ATL_DATA.routes` | City of Atlanta GIS Bicycle Routes layer |
| `neighborhoods.js` | `window.ATL_DATA.neighborhoods` | City of Atlanta GIS Neighborhood boundaries |
| `safety.js` | `window.ATL_DATA.crashes` / `.corridors` / `.stats` | GDOT crash CSV export (or NHTSA FARS with `--fars`) |

Commit them — GitHub Pages serves them as static assets next to `index.html`.
